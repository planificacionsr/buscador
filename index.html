Configuración necesaria:

  1. Publica tu Google Sheet como una página web: Archivo → Publicar en la web
  2. Copia el ID de tu hoja de cálculo de la URL (el código entre /d/ y /edit)
  3. Reemplaza el valor de SHEET_ID en el código JavaScript con tu ID
  4. Asegúrate de que tu hoja tenga las columnas: B (Fecha), C (Nombre), D (DNI), F (Localidad), G (Recurso)
